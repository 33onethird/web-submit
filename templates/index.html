33 ⅓ is a project by students at the WU Vienna which allows you to analyse Android APKs files and check if they are malware or not. What can you do on this site?

  • Step 1: upload an APK. Make sure it's a valid file
  • Step 2: if we already analysed this file, you will get a cached result. You can manually re-trigger an analysis if needed.
  • Step 3: the result of the classification will appear. A link to virustotal is provided for comparison purposes.
Should you find bugs, please create an issue on github. Thanks!
